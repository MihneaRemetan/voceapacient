# Vocea Pacientului

## Descriere generală
**Vocea Pacientului** este o aplicație web dezvoltată cu scopul de a oferi o platformă prin care utilizatorii pot publica și consulta experiențe legate de sistemul medical. Aplicația urmărește facilitarea exprimării opiniilor pacienților într-un cadru structurat, cu accent pe organizarea informației și pe separarea clară a responsabilităților între componentele aplicației.

Proiectul este realizat folosind o arhitectură de tip **client–server**, cu un frontend separat de backend, autentificare bazată pe token (JWT) și o bază de date relațională.

---

## Scopul aplicației
- Crearea unei platforme pentru publicarea experiențelor pacienților
- Implementarea unui sistem de gestionare a utilizatorilor (înregistrare și autentificare)
- Gestionarea și afișarea postărilor
- Separarea clară între frontend și backend
- Asigurarea unei baze solide pentru extinderi viitoare (moderare, roluri, atașamente etc.)

---

## Stadiul actual al dezvoltării
Aplicația se află în prezent **în curs de dezvoltare**.  
În această etapă:
- structura principală a aplicației este implementată;
- backend-ul este configurat și funcțional;
- frontend-ul este conectat la backend;
- există încă erori și funcționalități aflate în proces de stabilizare.

Dezvoltarea va continua în perioada următoare, cu accent pe:
- corectarea erorilor existente;
- îmbunătățirea interfeței utilizator;
- completarea funcționalităților planificate;
- testare și optimizare.

---

## Deployment
Pentru acest proiect sunt configurate următoarele platforme de deployment:
- **Vercel** – utilizat pentru partea de frontend;
- **Render** – utilizat pentru partea de backend.

Deployment-ul actual are rol de testare și dezvoltare, urmând să fie rafinat pe măsură ce aplicația evoluează.

---

## Notă
Acest proiect nu reprezintă versiunea finală a aplicației. Codul sursă și funcționalitățile vor fi extinse și îmbunătățite în etapele următoare de dezvoltare.
